# Dial Micrometer Deflection Simulator — PWA

Progressive Web App for training on dial micrometer offset calibration and main engine crankshaft/web deflection.

## Included

- `index.html` main simulator
- `manifest.webmanifest` PWA manifest
- `sw.js` service worker for offline caching
- `offline.html` offline fallback
- `icons/` PWA icons
- `docs/` Word training document

Deploy the files to GitHub Pages and open the page in a browser to install as a PWA.
