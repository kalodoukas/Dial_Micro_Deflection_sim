# Dial Micrometer Deflection Simulator — PWA

Εκπαιδευτική εφαρμογή PWA για εξομοίωση dial micrometer, offset calibration και crankshaft/web deflection κύριας μηχανής.

## Αρχεία

- `index.html` — κύρια εφαρμογή
- `manifest.webmanifest` — PWA manifest
- `sw.js` — service worker για offline cache
- `offline.html` — offline fallback
- `icons/` — PWA icons
- `docs/` — Word αρχείο με προβλήματα κύριας μηχανής / deflection

## Χρήση στο GitHub Pages

1. Ανέβασε όλα τα αρχεία στο root του repository.
2. Πήγαινε: **Settings → Pages**.
3. Source: **Deploy from branch**.
4. Branch: `main`, folder: `/root`.
5. Άνοιξε το URL του GitHub Pages.
6. Στον browser πάτησε **Install app** ή **Add to Home Screen**.

## Σημείωση

Για να ενεργοποιηθεί το PWA install/offline mode, η εφαρμογή πρέπει να τρέχει από HTTPS ή localhost. Το GitHub Pages παρέχει HTTPS.
